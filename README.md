# Prj_Heranca_Polimorfismo
Projeto demonstrando os conceitos de herança e polimorfismo na programação orientada a objetos. Apresenta uma hierarquia de classes, reutilização de código e métodos sobrescritos para comportamentos diferentes entre objetos, ilustrando na prática como esses princípios tornam o código mais organizado, reutilizável e flexível.
